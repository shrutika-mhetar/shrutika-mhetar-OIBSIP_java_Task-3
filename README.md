# OIBSIP Java Development Internship  
## Task 3 – ATM Interface

### Project Description
This project is a simulation of an **ATM Interface** developed using Java.  
It allows users to perform basic banking operations such as withdrawing money, depositing money, checking account balance, and viewing transaction history.

The project includes a **Graphical User Interface (GUI)** built using **Java Swing** and demonstrates important **Object-Oriented Programming (OOP)** concepts.

---

## Objective
The objective of this task is to understand how an ATM system works and implement a simplified ATM interface using Java programming.

---

## Features
- ATM Card Insert Simulation
- Login using Card Number and PIN
- Deposit Money
- Withdraw Money
- Check Account Balance
- Transaction History
- GUI Interface using Java Swing

---

## Technologies Used
- Java
- Java Swing
- Object-Oriented Programming (OOP)
- GitHub for version control

  ---
